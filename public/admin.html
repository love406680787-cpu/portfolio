GitHub 配置

配置 GitHub Personal Access Token，用于上传文件到仓库

如何获取 Token：
1. 打开 GitHub Settings → Personal Access Tokens
2. 点击「Generate new token (classic)」
3. 设置名称，过期时间选「No expiration」
4. 勾选权限：repo (Full control of private repositories)
5. 点击 Generate，复制生成的 Token 粘贴到下方
Token 只会保存在本地浏览器中，不会发送到任何第三方服务器

使用说明

部署步骤：

第一步：在 GitHub 创建新仓库（建议设为 Private 私有），仓库名比如 portfolio

第二步：在「设置」页面配置好上面的 GitHub 信息

第三步：首次配置后，后台会自动创建 data/works.json 文件

第四步：把整个项目文件夹推送到 GitHub 仓库

第五步：在 GitHub 仓库 Settings → Pages → Source 选择 main 分支和 / (root) 目录，保存后即可获得网址
