GitHub 核心配置

配置 API 令牌以授权后台修改远端数据库。

如何获取 Token：
1. 访问 GitHub Settings → Tokens
2. 点击「Generate new token (classic)」
3. 过期时间建议选择「No expiration」
4. 必须勾选 repo 权限
5. 生成后将内容粘贴至下方
安全提示：Token 仅通过浏览器直接发送至 GitHub API，不在任何中转服务器留存。
